# Glosarium Singkat

- **Akun**: entitas login pengguna di sistem (tabel `users`).
- **Pegawai**: pegawai outsource yang menjadi pengguna sistem.
- **Data master pegawai**: data identitas pegawai di tabel `employees`.
- **Kode pendaftaran/undangan**: kode yang digunakan untuk mendaftar akun baru.
- **Dashboard Pegawai**: halaman utama untuk role pegawai.
- **Role pegawai**: peran akun dengan akses pegawai (`employee`).
- **File yatim**: file yang tersimpan tanpa referensi data (orphan file).
- **Alur pendaftaran**: rangkaian proses dari input pengguna hingga akun dibuat.
- **Aturan bisnis**: kebijakan/ketentuan pendaftaran yang wajib dipenuhi.
- **Kuota kode**: batas penggunaan kode pendaftaran/undangan.
